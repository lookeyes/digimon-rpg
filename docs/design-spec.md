## 按钮规范

- 主按钮：渐变蓝底 + 圆角25px + 外发光
- 次按钮：暗色卡片底 + 蓝色边框
- 危险按钮：渐变红底
- 按下态：scale(0.96) 缩小
- 禁用态：opacity 0.5 + cursor not-allowed

## 卡片规范

- 背景: `--bg-card`
- 圆角: `--radius` (12px)
- 边框: 1px solid `--border`
- 阴影: `--shadow`
- 内边距: 16px
- 稀有卡片可加对应颜色的边框和外发光

## 字体

- 主字体: 'Noto Sans SC'（谷歌中文字体）
- Logo: 900粗度 + 渐变色
- 标题: 22px, 700粗度
- 正文: 14-16px, 400-500粗度
- 小字: 11-12px
- 强调数字: 28px, 900粗度

## 页面布局

- 最大宽度: 480px（手机适配）
- 居中显示
- 底部导航: 固定于底部, 高度约60px
- 内容区: 可滚动，底部留80px避开导航栏
- 安全区: 使用 env(safe-area-inset-bottom)

## 动效

- 按钮按下: scale 0.96, 0.2s
- 页面切换: 无特殊动效（保持简单）
- 扭蛋动画: CSS keyframes（震动、发光、裂开）
- 孵化动画: CSS keyframes（晃动、裂开、弹出）
