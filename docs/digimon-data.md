## 领域克制（双环设计）

**物性环：** 自然精灵 → 深海救星 → 龙之咆哮 → 金属帝国 → 丛林奇兵 → 自然精灵

**灵性环：** 病毒克星 → 噩梦士兵 → 风之守卫 → 黑暗区域 → 未知领域 → 病毒克星

克制方造成 **1.5x** 伤害，被克制方造成 **0.75x** 伤害。

若双方各有克制/被克制关系，则抵消为 **1.0x**。

## 种族（5种）

| 种族 | Emoji | 颜色 | 克制关系 |
|------|-------|------|----------|
| 疫苗 💉 | 绿色 #4ecca3 | 克病毒，被数据克 |
| 数据 💾 | 青色 #00d4ff | 克疫苗，被病毒克 |
| 病毒 🦠 | 红色 #e94560 | 克数据，被疫苗克 |
| 自由 🕊️ | 金色 #ffd700 | 无克制关系 |
| 未知 ❓ | 紫色 #b44dff | 与所有种族互克（1.2x） |

自由与任何种族无克制关系（1.0x）。
未知打所有种族 1.2x，所有种族打未知也 1.2x。

## 数码宝贝模板结构

```javascript
{
  id: 'agumon',
  name: '亚古兽',
  stage: '成长期',
  fields: ['nature_spirits', 'dragons_roar'],  // 所属领域（可多个）
  type: '疫苗',                                  // 种族
  description: '...',
  baseHp: 100, baseAtk: 25, baseDef: 15, baseSpd: 18,
  evolutionTree: [
    {
      name: '暴龙兽',
      stage: '成熟期',
      method: 'level',
      condition: 16,
      fields: ['dragons_roar', 'nature_spirits'],
      type: '疫苗',
      fieldExpRequired: { dragons_roar: 50 }     // 分支进化领域经验条件
    }
  ]
}
```

## 领域经验系统

- 每只数码宝贝在孵化时初始化 `fieldExp`（10领域，全部为0）
- 打败某数码兽时，按对方所属领域给参战宝贝分配该领域经验
- 经验分配量：10 / 对方领域数（向上取整），每个领域均分
- 分支进化可能要求特定领域的经验达标
