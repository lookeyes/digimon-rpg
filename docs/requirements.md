# 数码宝贝RPG — 项目需求文档

## 项目概述

一款运行在安卓手机上的数码宝贝题材RPG回合制游戏，玩家通过扭蛋获得数码蛋，孵化后培养数码宝贝进行战斗。

## 参考游戏

- 宝可梦系列（回合制战斗、收集养成）
- 数码宝贝：时空异客

## 核心系统

### 1. 账号系统
- 注册/登录（Bmob后端）
- 存档和进度云端保存
- 玩家资源：扭蛋币、金币

### 2. 数码宝贝获取
- 通过扭蛋获得数码蛋
- 消耗扭蛋币抽蛋
- 四种稀有度：普通（60%）、稀有（25%）、史诗（12%）、传说（3%）
- 支持单抽和十连抽（十连保底至少1个稀有+）
- 数码蛋有孵化倒计时，时间到后可孵化

### 3. 数码宝贝进化
- 等级自动进化
- 消耗特定道具进化
- 支持退化，回到之前形态
- 多分支进化路线（一只可选不同路线）

### 4. 战斗系统
- 3v3 回合制（3只同时上场）
- 属性克制机制
- PVE（打电脑）和 PVP（玩家对战）

### 5. 道具系统
- 各类道具（进化道具、回复道具、扭蛋币等）
- 背包管理

### 6. 交易场
- 玩家之间挂单买卖道具
- 自由定价

## 画面风格

- 2D立绘 + UI界面，偏卡牌游戏风格
- 暗黑数码主题配色
- 无地图走动，纯界面操作

## 平台

- 目标：安卓手机
- 技术：HTML5单页应用 → Capacitor打包APK
